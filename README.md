## Название: веб-приложение аенды коммеческой недвижимости ООО "ЭКО-ЖИЛКОМ-ДМИТОВ" 
### О проекте: Веб-приложение разработано в рамках дипломной работы по спициальности 09.02.07 " Информационные системы и программирование". Цель- создание малозатратного сайта для продвижения аренды офисов, складов и торговых полощадей в г.Дмитрове.
### Используемые технологии: Платформа: WordPress.com (бесплатный тариф). Верстка:HTML5,CSS3 (пользовательские стили).Интерактивность: JavaScript( аккордеон FAQ) . Карты: Яндекс.Карта (ifram). Форма: Встроенный блок WordPress.com
